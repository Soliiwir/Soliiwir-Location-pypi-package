# Location-Midterm
This Python package contains the code from my midterm assignment.  
It allows you to manage locations by adding, displaying, and searching saved locations.

# Features
    * load locations form Location.txt
    * Save new locations
    * Search locations by name 
    * Display all saved locations


# Installation
from project root run:
    pip install .\dist\location_package-0.1.0-py3-none-any.whl

Requirements
    * Python 3.8 +
